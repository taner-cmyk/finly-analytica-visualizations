# Finly Analytica - Professional Visualization Suite

This repository contains a suite of high-performance community visualizations for **Looker Studio**, developed by **Dataconnecta**. These components are specifically engineered for advanced financial analysis, real-time market tracking, and interactive reporting.

## 🚀 Key Components

### 1. Financial Heatmap
An interactive treemap for market sectors and stock performance. Features dynamic color scaling based on financial performance metrics.

### 2. Financial Marquee (Kayan Bant)
A high-performance horizontal scrolling band designed for real-time stock prices and indices. Optimized for 1-Dimension and 1-Metric data structures.

### 3. Interactive Scorecard
A dual-metric scorecard that displays dimension labels and percentage changes simultaneously. Supports interactive filtering across the dashboard.

### 4. Sequential Comparison Table (QoQ)
Optimized for Balance Sheets and sequential analysis. Automatically compares the current period with the immediate preceding period.

### 5. YOY Comparison Table (Yearly)
Designed for Income Statements and growth tracking. Automatically aligns data with the same period from the previous year.

## 🛠 Deployment & Technology
- **Host:** Google Cloud Storage (GCS)
- **Library:** DSCC (Looker Studio Community Component Library)
- **Focus:** Financial accuracy, responsive design, and low latency.
